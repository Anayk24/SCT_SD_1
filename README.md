# Java Temperature Converter

A simple **Java console application** that converts temperatures between **Celsius, Fahrenheit, and Kelvin** scales.  
This program lets users choose a conversion type from a menu and enter a value to convert, then displays the result.

##  Features

- Converts between all three major temperature units:  
  ✔ Celsius → Fahrenheit & Kelvin  
  ✔ Fahrenheit → Celsius & Kelvin  
  ✔ Kelvin → Celsius & Fahrenheit
- Interactive menu for easy use  
- Developer-friendly modular code
